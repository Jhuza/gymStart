# Stack Tecnológico: GymStart

## Fase 1 (Actual) — Maquetación y Documentación

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura semántica (header, main, section, article) |
| CSS3 | Estilos, Flexbox, Grid, variables CSS |
| Git + GitHub | Control de versiones y repositorio remoto |

## Fase 2 (Próximamente) — Lógica e Interactividad

| Tecnología | Uso |
|---|---|
| JavaScript Vanilla | Lógica, manipulación del DOM, eventos |
| LocalStorage | Persistencia de datos sin backend |
| DOM inputs | Captura de datos del usuario en tiempo real |

## Fase 3 (Final) — Componentización y Despliegue

| Tecnología | Uso |
|---|---|
| React | Componentización de la interfaz |
| Firebase Firestore | Base de datos en la nube |
| Firebase Auth | Autenticación de usuarios |
| Vercel o Netlify | Despliegue y hosting gratuito |

## Herramientas de Desarrollo

| Herramienta | Uso |
|---|---|
| VS Code | Editor de código principal |
| Git Bash | Terminal para control de versiones |
| Google Fonts | Fuente Inter para tipografía |
| Lucide Icons | Iconografía de línea fina |
| Stitch / v0.dev | Prototipado visual con IA |

## Decisiones Técnicas
- **Mobile First:** Todo el CSS se escribe primero para móvil y luego se adapta a desktop
- **Sin frameworks en Fase 1:** HTML y CSS puros para dominar los fundamentos
- **LocalStorage en Fase 2:** Permite guardar datos sin necesitar un servidor
- **Firebase en Fase 3:** Escalable, gratuito para proyectos pequeños y fácil de integrar con React
