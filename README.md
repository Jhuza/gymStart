# gymStart 🏋️

Aplicación web para jóvenes que están comenzando su camino en el fitness.

## Descripción
GymStart permite llevar un registro diario de ejercicios, hidratación y alimentación en un solo lugar, con una interfaz limpia que no abruma al usuario principiante.

## Usuario Objetivo
Jóvenes entre 16 y 28 años que empiezan en el gym y necesitan una herramienta simple, visual y motivadora.

## Cómo ejecutarlo
1. Clona el repositorio: `git clone https://github.com/Jhuza/gymStart.git`
2. Abre el archivo `index.html` en tu navegador

## Stack Tecnológico
- HTML5 + CSS3
- JavaScript (Fase 2)
- React + Firebase (Fase 3)
